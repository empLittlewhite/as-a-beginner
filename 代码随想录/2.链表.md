	链表的定义


```cpp
struct ListNode{
	int val;
	ListNode *next;
	//还得写一下构造函数吗？可以有的
	//这是什么写法cpp11里面的吗？
	ListNode(int x):val(x),next(NULL) {}

};


//补充，cpp11中有自带的构造函数，
//但是不会初始化成员变量
1.这是已经定义的构造函数
ListNode* head = new ListNode(5);

2.这是系统自带的
ListNode* head = new ListNode();
head->val = 5;


```
